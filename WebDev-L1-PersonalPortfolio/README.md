# Personal Portfolio — RK.

**Track:** Web Development | **Level:** 1 (Task 2 of 3) — Personal Portfolio

## Overview
A single-page personal portfolio site built with HTML5 and CSS3 (vanilla JS for interactivity — no frameworks). It serves as a digital résumé: hero/profile section, about, skills, projects, certifications, and contact.

## Features
- Hero section with name, role title, and avatar placeholder
- About Me section
- Skills grid grouped by category (languages, AI/ML, core CS, tools)
- Project cards linking to live GitHub repositories
- Certifications log
- Contact section with email, LinkedIn, and GitHub
- Smooth-scroll navigation with active-link highlighting
- Fully responsive layout (desktop + mobile, with a mobile nav toggle)
- Light entrance/hover animations (typing effect, scroll-reveal, hover states)

## Tech Stack
- HTML5
- CSS3 (custom properties, Flexbox/Grid, no framework)
- Vanilla JavaScript (typing effect, IntersectionObserver for scroll-reveal and active nav state, mobile menu toggle)
- Google Fonts: Syne (display), Inter (body)

## How to Run
Open `index.html` directly in any browser — no build step or dependencies required.
