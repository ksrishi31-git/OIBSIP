A bit about me

I'm an AI/ML learning student who likes turning ideas into things that actually run — from a driver-monitoring system built on OpenCV and MediaPipe to a tool that sorts documents using AI. I spend most of my time experimenting with Python, computer vision and prompt engineering, while steadily working through the fundamentals: data structures, algorithms and core ML concepts.

Outside coursework, I'm drawn to problems with a real-world edge — road safety, food waste, document chaos — and I'd rather ship a rough working prototype than leave an idea on paper. I'm currently deepening my foundation through structured certifications in Generative AI, prompt engineering and applied Python.
