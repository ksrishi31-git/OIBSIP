# Temperature Converter

**Track:** Web Development | **Level:** 1 (Task 3 of 3) — Temperature Converter

## Overview
A single-page temperature conversion tool that converts between Celsius, Fahrenheit, and Kelvin, with input validation and a breakdown showing all three units at once.

## Features
- Numeric input with unit selector (Celsius / Fahrenheit / Kelvin)
- Input validation with inline error message for empty/invalid input
- Converts and displays the result in all three units simultaneously
- Reset button to clear the form
- Enter-key support to trigger conversion
- Reference info cards (freezing point, boiling point, body temperature)
- Fully responsive layout

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (DOM manipulation, event listeners, conversion logic)

## How to Run
Open `index.html` directly in any browser — no build step or dependencies required.

## Conversion Formulas Used
- °F = (°C × 9/5) + 32
- K = °C + 273.15
- °C = (°F − 32) × 5/9
