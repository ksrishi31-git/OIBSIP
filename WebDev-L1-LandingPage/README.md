# Landing Page — WanderIndia

**Track:** Web Development | **Level:** 1 (Task 1 of 3) — Landing Page

## Overview
A responsive travel-agency landing page for a fictional brand, "WanderIndia," promoting curated trip packages across India.

## Features
- Navbar with logo and anchor navigation
- Hero section with headline, subtext, and dual CTA buttons
- Stats bar (destinations, travellers, years of experience, rating)
- Popular destinations section with pricing cards (Kerala, Rajasthan, Kashmir)
- "Why Choose Us" feature grid
- "How It Works" 4-step process section
- Testimonials section with star ratings
- Newsletter signup section
- Multi-column footer with site links

## Tech Stack
- HTML5
- CSS3 (Flexbox-based layout, no framework)

## How to Run
Open `index.html` directly in any browser — no build step or dependencies required.

## Notes
- All destination images are emoji-based colour-block placeholders — swap in real photography before final submission if your Task Card requires actual images.
- "Book Now" and footer links are placeholders (`#`) and not yet wired to a booking flow.
